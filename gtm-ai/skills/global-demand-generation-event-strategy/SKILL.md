Strategic Event Framework:

1. Event Portfolio Design
Create a multi-format event strategy encompassing:

- Virtual events (webinars, workshops, summits)
- Field and ABM-style targeted events
- Customer-exclusive events
- Branded global conference

Strategic Alignment:
Align event program with key company priorities:
- Enterprise growth acceleration
- Category and brand awareness
- Product-led and self-serve adoption strategies

2. Regional Event Prioritization
Customize event approach for key regions:
- North America: Comprehensive multi-format approach
- EMEA: Executive and partner-focused events
- APAC: Virtual and startup-centric activations
- LATAM: Community and partner-driven engagement

3. Event Type Objectives Breakdown
For each event type, define specific goals:
a) Physical Events
- Generate brand awareness
- Create pipeline opportunities
- Establish ecosystem presence

b) Virtual Events
- Provide scalable education
- Expand use case visibility
- Facilitate product training and onboarding

c) ABM Field Events
- Enable high-touch conversions
- Accelerate deal cycles
- Drive account expansion

d) Customer-Specific Events
- Support upsell strategies
- Build customer advocacy
- Improve net revenue retention (NRR)

e) Global Conference
- Establish thought leadership
- Create category narrative
- Reinforce company platform positioning

ABM and Field Event Approach:

Customer Engagement Events:

- Customer success forums

- Peer networking sessions

5. Partner Co-Hosting Strategy

- Cloud/tech partner events

- Physical events: Leads, trials, sourced pipeline
